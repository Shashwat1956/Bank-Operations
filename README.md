<h1>Bank Operations</h1>
<br>
<p>This repository contains a C++ program for managing bank operations. Users can create accounts, 
    deposit and withdraw money, check balances, and display account information. 
    The program ensures account number uniqueness and validates user inputs.</p>

<br>
<h3>Features</h3>
<ul>
    <li><b>Create Account:</b>Add new bank accounts with a unique account number 
        and account holder's name.</li><br>
    <li><b>Deposit:</b>Deposit a specified amount into an existing account.</li><br>
    <li><b>Withdraw:</b>Withdraw a specified amount from an existing account, ensuring sufficient balance.</li><br>
    <li><b>Check Balance:</b>View the current balance of a specified account.</li><br>
    <li><b>Display All Accounts:</b>Display information for all existing accounts.</li>
</ul>
